## Run Locally

Clone the project

```bash
  https://github.com/souravsuvarna/QrTalk.git
```

Go to the project directory

```bash
  cd QrTalk
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
